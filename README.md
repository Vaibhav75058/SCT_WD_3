A modern, responsive, and interactive **Tic-Tac-Toe Web Application** built using **HTML, CSS, and Vanilla JavaScript**.

It includes both **Player vs Player** and **Unbeatable AI (Minimax Algorithm)** modes with a clean neumorphism-inspired UI and dark mode support.

## ✨ Features

- 🎯 Player vs Player mode  
- 🤖 Player vs Computer (Unbeatable AI – Minimax Algorithm)  
- 🌙 Light / Dark theme toggle  
- 📊 Live scoreboard tracking  
- 🏆 Automatic win detection with highlighted winning cells  
- 🔁 Restart game & Reset scores  
- 📱 Fully responsive design  
- ✨ Smooth animations and interactive UI  

---

## 🧠 AI Logic

The computer opponent uses the **Minimax Algorithm**, ensuring:

- Optimal move selection  
- No losing scenarios  
- Intelligent board evaluation  

This makes the AI **mathematically unbeatable**.

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling (Flexbox, Grid, CSS Variables, Neumorphism)  
- **JavaScript (ES6)** – Game logic, DOM manipulation, AI
